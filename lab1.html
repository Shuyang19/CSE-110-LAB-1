Section 1

Make the entire navbar “sticky” so it will stay on the top even when scrolling down (Hint: refer to https://www.w3schools.com/howto/howto_js_navbar_sticky.asp). Note that the anchor links won’t scroll to the right location anymore, since it positions to the top of the sticky navbar. Thus we need to add a scroll-margin-top for each section to offset the scroll/jump position on the window – set it to be the height of the sticky navbar. For now you can hard-code it based on the computed height of your navbar (check the inspector tool!).
